## 📝 文章写作

### 通用模板
```markdown
你是一位专业的 [领域] 作者。

**主题**：[文章主题]

**目标读者**：[读者画像]

**文章目的**：
- [ ] 科普教育
- [ ] 观点表达
- [ ] 产品推广
- [ ] 经验分享
- [ ] 其他：[说明]

**风格要求**：
- 语气：[正式/轻松/幽默/严肃]
- 长度：[字数范围]
- 结构：[列表/故事/问答/其他]

**必须包含**：
- [要点 1]
- [要点 2]
- [要点 3]

**避免**：
- [禁忌 1]
- [禁忌 2]

请输出完整文章。
```

## 📧 邮件写作

### 商务邮件
```markdown
**邮件类型**：
- [ ] 合作邀约
- [ ] 跟进询问
- [ ] 问题反馈
- [ ] 会议邀请
- [ ] 感谢致意

**收件人**：[职位/关系]

**核心目的**：[一句话]

**关键信息**：
- [信息 1]
- [信息 2]

**期望行动**：[希望对方做什么]

**语气**：正式但不生硬

请提供：
1. 3 个主题行选项
2. 完整邮件正文
3. 跟进建议 (如无回复)
```

### 模板示例
```markdown
**邮件类型**：合作邀约

**收件人**：某公司商务负责人

**核心目的**：邀请对方参加我们的技术沙龙并分享

**关键信息**：
- 活动时间：3 月 20 日，北京
- 预期观众：200+ 技术从业者
- 我们提供：差旅 + 酬劳 + 曝光

**期望行动**：确认意向并安排初步沟通

请写一封得体且有吸引力的邀约邮件。
```

## 📱 社交媒体文案

### 通用模板
```markdown
**平台**：[微博/微信/LinkedIn/Twitter/小红书]

**主题**：[内容主题]

**目标**：
- [ ] 增加关注
- [ ] 引导点击
- [ ] 促进互动
- [ ] 品牌曝光

**内容要点**：
- [要点 1]
- [要点 2]

**风格**：[幽默/专业/感性/励志]

**必须包含**：
- [ ] 话题标签
- [ ] 行动号召
- [ ] 表情符号

请提供 3 个版本供选择。
```

### 平台特性参考
| 平台 | 特点 | 长度 | 风格 |
|------|------|------|------|
| 微博 | 热点驱动 | 140 字 | 轻松、有梗 |
| 微信 | 深度阅读 | 不限 | 专业、有温度 |
| LinkedIn | 职业社交 | 300-500 字 | 专业、有价值 |
| 小红书 | 种草分享 | 300-800 字 | 真实、有细节 |
| Twitter | 即时动态 | 280 字 | 简洁、有观点 |

## 🔍 SEO 优化写作

### 模板
```markdown
**主题**：[文章主题]

**目标关键词**：
- 主关键词：[关键词]
- 长尾关键词：[列表]

**搜索意图**：[信息型/交易型/导航型]

**竞争对手**：[可选，对标文章]

**要求**：
- 自然融入关键词
- 标题包含主关键词
- H2/H3 合理使用
- 内部链接建议
- Meta 描述

请输出 SEO 优化的完整文章。
```

## 📋 内容大纲

### 模板
```markdown
**主题**：[文章/内容主题]

**目的**：[写作目的]

**读者**：[目标读者]

**长度**：[预计字数]

请提供详细大纲，包含：
1. 标题选项 (3 个)
2. 引言思路
3. 主体部分 (H2/H3 结构)
4. 每个小节的关键点
5. 结论/行动号召

大纲确认后我再让你展开写作。
```
