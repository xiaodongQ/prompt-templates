# 💻 编程开发提示词模板

---

## 📝 代码生成

### 基础模板
```markdown
你是一位资深 [语言/框架] 工程师。

**任务**：编写一个 [功能描述]

**要求**：
- 使用 [语言版本/框架版本]
- 遵循 [编码规范，如 PEP8/Google Style]
- 包含完整的错误处理
- 添加必要的注释
- 考虑边界情况

**输出**：
1. 完整可运行的代码
2. 使用说明
3. 测试示例
```

### 完整示例
```markdown
你是一位资深 Python 工程师。

**任务**：编写一个文件批量重命名工具

**要求**：
- Python 3.10+
- 递归处理子目录
- 支持正则表达式匹配
- 先预览变更，确认后执行
- 添加日志记录
- 处理权限错误等异常

**输出**：
1. 完整代码 (单文件，可直接运行)
2. 命令行使用示例
3. 3 个典型使用场景
```
