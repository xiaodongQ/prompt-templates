## 🎯 目标设定

### SMART 目标
```markdown
**目标领域**：[工作/学习/健康/其他]

**初步想法**：[模糊的目标]

**请用 SMART 原则帮我完善**：

- **S**pecific (具体)：目标具体明确吗？
- **M**easurable (可衡量)：如何衡量进展？
- **A**chievable (可实现)：现实可行吗？
- **R**elevant (相关)：与大方向一致吗？
- **T**ime-bound (有时限)：截止日期是什么？

请帮我：
1. 将模糊目标转化为 SMART 目标
2. 拆解为阶段性里程碑
3. 设计进度追踪方法
4. 预判可能的障碍及应对
```

## 📊 优先级排序

### 艾森豪威尔矩阵
```markdown
**待办任务列表**：
1. [任务 1]
2. [任务 2]
3. [任务 3]
...

**请帮我按艾森豪威尔矩阵分类**：

| | 紧急 | 不紧急 |
|---|---|---|
| **重要** | 第一象限：立即做 | 第二象限：计划做 |
| **不重要** | 第三象限：委托做 | 第四象限：不做/少做 |

对每个任务：
1. 判断重要程度 (1-10)
2. 判断紧急程度 (1-10)
3. 放入对应象限
4. 给出处理建议

请分类并说明理由。
```

### RICE 评分
```markdown
**候选项目**：
- [项目 A]
- [项目 B]
- [项目 C]

**请用 RICE 模型评分**：

| 项目 | Reach (影响人数) | Impact (影响程度 1-5) | Confidence (信心 0-100%) | Effort (工作量) | RICE 分数 |
|------|------------------|----------------------|-------------------------|-----------------|-----------|

**评分标准**：
- Reach：影响的用户/客户数量
- Impact：对每个人的影响程度 (0.25/0.5/1/2/3)
- Confidence：对估算的信心
- Effort：人周/人月

请评分并排序。
```
