# 📚 学习理解提示词模板

---

## 🎓 概念解释

### 费曼技巧版
```markdown
我想理解 [概念/主题]。

**我的背景**：
- 熟悉：[相关知识]
- 不熟悉：[相关知识]

**请用费曼技巧解释**：
1. 用最简单的语言解释核心概念
2. 用生活中的类比帮助理解
3. 指出常见的误解
4. 给我一个小测试检验理解

**避免**：
- 专业术语堆砌
- 过度抽象

请像给聪明的 12 岁孩子讲解一样。
```

### 渐进式解释
```markdown
请分层次解释 [概念]：

**Level 1 - 小白版** (1 句话)
[一句话核心]

**Level 2 - 入门版** (100 字)
[基本解释 + 类比]

**Level 3 - 进阶版** (300 字)
[技术细节 + 应用场景]

**Level 4 - 专家版**
[深入分析 + 前沿发展]

我从 [Level X] 开始，根据需要深入。
```

## 🧠 知识图谱

### 模板
```markdown
**主题**：[知识领域]

**请帮我构建知识图谱**：

1. **核心概念** (必须掌握的 5-10 个概念)

2. **概念关系** (用文字描述关联)
   - A 是 B 的基础
   - C 和 D 是并列的两种方法
   - E 是 F 的应用场景

3. **学习顺序** (先后依赖关系)

4. **实践节点** (每个阶段可以做什么项目)

5. **常见误区** (学习过程中容易混淆的地方)

请用结构化方式呈现，帮助我建立整体认知。
```

## ❓ 问答学习

### 苏格拉底式教学
```markdown
我想通过问答学习 [主题]。

**规则**：
- 你不要直接给答案
- 通过提问引导我思考
- 根据我的回答调整难度
- 最后帮我总结

**我的水平**：[描述]

**学习目标**：[描述]

请开始第一个问题。
```

### 深度问答
```markdown
关于 [主题]，我有以下问题：

**基础问题**：
1. [问题 1]
2. [问题 2]

**深入问题**：
1. [问题 1]
2. [问题 2]

**应用问题**：
1. [问题 1]
2. [问题 2]

请逐一详细解答，并：
- 提供示例
- 指出相关概念
- 推荐延伸阅读
```

## 🔁 复习巩固

### 间隔重复卡片
```markdown
**主题**：[学习内容]

**请帮我生成复习卡片**：

**格式**：
```
问题：[问题]
答案：[答案]
难度：[简单/中等/困难]
关联：[相关知识点]
```

**要求**：
- 覆盖核心概念
- 问题形式多样 (选择/填空/简答)
- 共 [数量] 张卡片
- 标注建议复习时间 (1 天/3 天/7 天/14 天)

请生成卡片。
```

```markdown
- 简答题：3 道
- 应用题：2 道

请先出题，我回答后再给答案和解析。
```

## 🧩 类比学习

### 模板
```markdown
**新概念**：[要学习的概念]

**我熟悉的领域**：[如"烹饪"/"开车"/"游戏"]

**请用我熟悉的领域来类比解释新概念**：

要求：
1. 找出两个领域的相似点
2. 用熟悉的事物解释陌生的
3. 指出类比的局限性 (哪里不像)

例如：
- 用"餐厅点餐"类比"API 请求"
- 用"水管系统"类比"电路"
- 用"图书馆"类比"数据库"

请帮我建立这个类比。
```

```markdown
请用对比表格 + 具体示例帮我理清区别。
```

## 🎯 学习反思

### 模板
```markdown
**学习主题**：[主题]

**学习时间**：[时长]

**请引导我反思**：

1. **收获**：我学到了什么最重要的东西？

2. **连接**：这些新知识和已有知识有什么联系？

3. **应用**：我可以在哪里应用这些知识？

4. **不足**：我还有哪些不理解的地方？

5. **下一步**：我应该继续学习什么？

请通过提问引导我完成反思，然后帮我总结。
```
