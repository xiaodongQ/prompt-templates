## ⏱️ 3 分钟快速上手

### 第 1 分钟：理解核心

**好提示词 = 把 AI 当聪明的实习生**

你不会对实习生说"帮我写个脚本"，你会说：
> "用 Python 写个脚本，把当前目录下所有.txt 文件改名，空格换下划线。先列出来让我确认，再执行。"

**就这么简单。**

---

### 第 2 分钟：用这个万能模板

```markdown
你是一位 [角色]，请帮我 [任务]。

**背景**：[相关情况]
**要求**：[具体约束]
**输出**：[期望格式]

如果有不清楚的，请先问我。
```

**填完就发，比你自己想省 80% 时间。**

---

### 第 3 分钟：看效果对比

#### ❌ 你以前可能这样问
```
帮我写个 Python 脚本
```
→ AI：请问需要什么功能的脚本？

#### ✅ 现在这样问
```markdown
你是一位资深 Python 工程师，请帮我写一个脚本。

**背景**：我有 1000+ 个文件要批量重命名
**任务**：把文件名中的空格替换为下划线，递归处理子目录
**要求**：先预览变更，我确认后再执行；添加错误处理
**输出**：完整代码 + 使用说明

如果有不清楚的，请先问我。
```
→ AI：直接输出完整可运行的代码

**看到了吗？这就是差距。**

## 🎯 记住这 5 个要点

```
□ 给背景 - AI 不知道你的上下文
□ 定任务 - 一句话说明要做什么
□ 说要求 - 约束条件越具体越好
□ 定格式 - 表格/列表/代码/文章
□ 留互动 - "不清楚先问我"
```

**就这 5 点，做到就已经超过 90% 的人了。**

## ❓ 常见问题

**Q：模板太麻烦，我不想每次都填**
> A：用 3 次就习惯了。想想你省下的追问时间。

**Q：我的任务很简单，也需要吗？**
> A：简单任务可以简化，但背景+ 任务 + 格式不能少。

**Q：AI 回答还是不好怎么办？**
> A：99% 的情况是提示词不够清晰。补充信息，再问一次。

**Q：要记住所有框架吗？**
> A：不用。万能模板够你用 80% 的场景。其他的当字典查。
