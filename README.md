## 🚀 快速开始

### 3 秒速查 - 提问前问自己

```
□ 我说明了任务背景吗？
□ 我定义了输出格式吗？
□ 我给出了约束条件吗？
□ 我提供了示例吗？
```

### 万能公式

```
[角色] + [任务] + [背景] + [要求] + [格式] + [示例]
```

### 示例对比

❌ **零散提问**
> "帮我写个 Python 脚本"

✅ **体系化提问**
> "你是一位资深 Python 工程师。我需要写一个脚本，批量重命名目录下的文件（将空格替换为下划线）。
> 
> **要求**：
> - 递归处理子目录
> - 先预览变更，确认后执行
> - 添加错误处理和日志
> 
> **输出**：完整可运行的代码 + 使用说明"

**效果差异**：❌ 需要 3-4 轮对话才能得到可用结果 | ✅ 一次得到完整方案

## 📖 核心理念

**提示词工程不是魔法，是沟通艺术。**

好的提示词 = 清晰的思考 × 结构化的表达

当你觉得 AI 回答不好时，通常不是 AI 的问题，而是：
1. 任务定义不够清晰
2. 缺少必要的上下文
3. 没有指定输出格式
4. 没有提供示例参考

本模板库的目标：**帮你把零散的想法，变成结构化的指令。**
