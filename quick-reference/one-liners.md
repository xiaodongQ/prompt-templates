# 💬 万能句式速查

> 复制粘贴就能用的高频提示词句式

---

## 🎭 角色设定

```
你是一位 [领域] 专家，拥有 [X 年] 经验。
```

```
假设你是 [角色]，你的特点是 [特点 1,2,3]。
```

```
请用 [职业/角色] 的视角来分析这个问题。
```

```
你是我信任的 [顾问/导师/伙伴]，请给我坦诚的建议。
```

---

## 📋 任务说明

```
请帮我完成以下任务：[任务描述]
```

```
我需要你帮我 [动词：写/分析/总结/设计/优化...] [对象]。
```

```
目标是 [目标]，请帮我达成。
```

```
请把这个复杂问题拆解成可执行的小步骤。
```

---

## 📝 格式要求

```
请用表格呈现，包含以下列：[列名]
```

```
请用 bullet points 列出关键点。
```

```
请用 JSON 格式输出，结构如下：[结构]
```

```
请按以下结构组织：1... 2... 3...
```

```
输出不超过 [X] 字/词/行。
```

```
请先给结论，再展开说明。
```

```
用 Markdown 格式输出。
```

---

## 🎨 风格要求

```
用通俗易懂的语言，避免专业术语。
```

```
用专业严谨的语气，像学术论文。
```

```
用轻松幽默的风格，像朋友聊天。
```

```
用简洁有力的表达，像广告文案。
```

```
用温暖鼓励的语气，像导师指导。
```

---

## 🧠 思考引导

```
请逐步思考，展示你的推理过程。
```

```
Let's think step by step.
```

```
在给出答案前，请先分析问题的关键点。
```

```
请从多个角度分析这个问题。
```

```
请列出至少 3 种可能的解决方案。
```

```
请评估每个方案的优缺点。
```

---

## 📚 学习理解

```
请用费曼技巧解释这个概念。
```

```
用生活中的类比帮我理解 [概念]。
```

```
假设我是 [年龄/水平]，请解释 [主题]。
```

```
我理解了 [A]，但不理解 [B]，请帮我连接这两个概念。
```

```
请出 3 道题测试我是否理解了。
```

---

## ✍️ 写作润色

```
请润色以下文字，使其更 [目标风格]。
```

```
请保持原意，让表达更简洁有力。
```

```
请检查语法错误和表达不当的地方。
```

```
请给这篇文章起 5 个吸引人的标题。
```

```
请帮我扩写/缩写到 [字数] 字。
```

---

## 🔍 分析总结

```
请用一句话总结核心观点。
```

```
请列出 3-5 个关键要点。
```

```
请提取最重要的信息，忽略细节。
```

```
请分析这段话的潜在含义。
```

```
请找出逻辑漏洞或假设。
```

---

## 💡 创意生成

```
请头脑风暴 10 个 [主题] 的想法。
```

```
请给我 3 个不同风格的方案供选择。
```

```
如果不考虑限制，理想的方案是什么？
```

```
请从反面思考这个问题。
```

```
请借鉴 [其他领域] 的做法。
```

---

## 🔄 迭代优化

```
这是第一版，请基于以下反馈优化：[反馈]
```

```
这个方向对了，请在 [方面] 再深入。
```

```
不太对，我实际想要的是 [说明]。
```

```
很好！请保持这个风格，继续完成 [其他部分]。
```

---

## ❓ 追问技巧

```
能详细解释一下 [具体点] 吗？
```

```
为什么你会得出这个结论？
```

```
还有其他可能的解释/方案吗？
```

```
如果 [条件变化]，结果会有什么不同？
```

```
这个方案的潜在风险是什么？
```

---

## 🎯 决策辅助

```
请列出做这个决定的利弊。
```

```
如果用 1-10 分评估，你会打几分？为什么？
```

```
请从短期/中期/长期分析影响。
```

```
请考虑 [利益相关者] 的视角。
```

```
最坏的情况是什么？如何应对？
```

---

## 📊 对比分析

```
请对比 A 和 B 的优缺点。
```

```
请用表格展示差异。
```

```
在 [场景 X] 下，哪个更合适？为什么？
```

```
从 [维度 1,2,3] 分别对比。
```

---

## 🛠️ 问题解决

```
问题的根本原因可能是什么？
```

```
请给出 3 种解决方案，按可行性排序。
```

```
如果资源无限，最佳方案是什么？
```

```
如果资源有限，最务实的方案是什么？
```

```
实施这个方案的第一步是什么？
```

---

## 📈 反馈评估

```
请客观评价这个 [作品/方案] 的质量。
```

```
最大的优点是什么？最大的不足是什么？
```

```
如果只能改一处，改哪里提升最大？
```

```
从 [目标用户] 的角度看，哪里需要改进？
```

---

## 🚀 行动导向

```
基于以上分析，我应该采取什么行动？
```

```
请给出具体可执行的下一步。
```

```
如果今天只能做一件事，做什么？
```

```
请设计一个 7 天行动计划。
```

---

## 🎬 场景模拟

```
让我们模拟一下这个场景：[描述]
```

```
如果你是 [角色]，你会怎么做？
```

```
预演一下可能的对话流程。
```

```
预测一下这个决定可能的结果。
```

---

## 🔐 安全检查

```
这个回答有什么潜在问题或风险吗？
```

```
我可能忽略了什么重要因素？
```

```
有什么假设需要验证？
```

```
在什么情况下这个建议不适用？
```

---

## 💬 对话管理

```
在开始前，你有什么需要我补充的信息吗？
```

```
我的问题清楚吗？有需要澄清的地方吗？
```

```
请确认你理解了我的需求，复述一下。
```

```
我们暂停一下，让我消化已有的信息。
```

---

## 🎁  bonus 句式

```
我知道这可能有点复杂，请耐心帮我分析。
```

```
我不确定这个问题是否清晰，如有需要请追问。
```

```
请像对待聪明但外行的朋友一样解释。
```

```
我时间有限，请先给最关键的 20%。
```

```
这是一个重要的决定，请帮我仔细分析。
```

```
请坦诚告诉我，不要迎合我的想法。
```

---

*提示：把这些句式加入书签，需要时快速组合使用*
