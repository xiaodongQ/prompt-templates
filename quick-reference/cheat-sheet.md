# 📇 提示词速查卡

> 一页纸核心要点，打印贴桌边

---

## 🎯 万能公式

```
[角色] + [任务] + [背景] + [要求] + [格式] + [示例]
```

**示例**：
```
你是一位资深 Python 工程师（角色），
帮我写一个文件批处理脚本（任务），
需要处理 1000+ 文件（背景），
要求递归处理、先预览后执行（要求），
输出完整代码 + 使用说明（格式），
类似这样：[示例代码]（示例）
```

---

## ✅ 提问前 5 问

```
□ 背景说了吗？
□ 任务清吗？
□ 格式定了吗？
□ 约束给了吗？
□ 示例有吗？
```

---

## 🔴 常见错误

| 错误 | 修复 |
|------|------|
| 太模糊 | 加具体约束 |
| 缺背景 | 补充上下文 |
| 无格式 | 指定输出形式 |
| 一次太多 | 分步进行 |
| 不追问 | 主动要求澄清 |

## 🎭 常用角色

```
• 资深 [领域] 工程师
• 专业 [类型] 作者
• 严格但善于指导的 [角色]
• 耐心的 [领域] 导师
• 有 15 年经验的 [职业]
```

---

## 📋 常用格式

```
• 表格：| 列 1 | 列 2 | 列 3 |
• 列表：- 要点 1\n- 要点 2
• 代码：```语言\n代码\n```
• JSON：{"key": "value"}
• 分级：## 标题\n### 子标题
```

---

## 🧠 思考引导

```
"请逐步思考，展示推理过程"
"Let's think step by step"
"从多角度分析这个问题"
"列出 3 种方案并对比"
"先给结论，再展开说明"
```

---

## 🔄 迭代话术

```
**方向对**："这个方向对了，请在 [方面] 深入"
**不对**："不太对，我实际想要的是 [说明]"
**补充**："补充一个信息：[新信息]"
**简化**："太复杂了，请用更简单的方式"
**继续**："很好，请继续完成 [其他部分]"
```

## 💡 黄金法则

```
1. 好提示词 = 清晰思考
2. 多花 1 分钟想，节省 10 分钟改
3. AI 不知道你的上下文，说出来
4. 一次一件事，质量更高
5. 不满意就追问，别凑合
```
